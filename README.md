## 使用

- 方向键 上：<kbd>↑</kbd> 上一个视频
- 方向键 下：<kbd>↓</kbd> 下一个视频
- 方向键 ← →：<kbd>←</kbd>, <kbd>→</kbd> 刷新
- 老板键 Alt + Q
